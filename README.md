# Практична робота на наслідування
## Дисклеймер
Я трохи пролажав непрочитавши на початку роботи повністю завдання та одумав, що треба ПРИДУМАТИ ВЛАСНИЙ КЛАС який буде наслідуватися далі і далі, але потім поняв шо буквально шо можна просто взяти клас таврини додати клас іншої тварини і всьо, але пізно, мені занадто жалко тому ось ми тут.
## Коротке пояснення теми
За тему я обрав транспорт. Вони бувають наземні, як от машини, повітряні, як літаки, та водні, різні кораблі та катери(їх я не добавив, хоча була ідея, але довго рутинна робота). Ми маємо клас Transport Який є головним описом усього транспорту, тут є такі зміні, як назва моделі, кількість пасажирів, та максимальна швидкість. Що до методів, то це запуск машини, зупинка транспорту, зміна швидкості та звісно що вивід інформації.
### Код класу Transport
```java
package domain;

public class Transport {
    protected String modelName;
    protected int crewCount;
    protected double maxSpeed;

    public Transport() {
        modelName="BMW";
        crewCount=4;
        maxSpeed=210;
    }

    public void move(){
        System.out.println("Transport is moving...");
    }
    
    public void changeSpeed(){
        System.out.println("Transport has just changed a speed...");
    }
    
    public void stop(){
        System.out.println("Transport isn`t moving anymore...");
    }

    
    @Override
    public String toString() {
        return "I have a transport, its model is "+modelName+", there are "+crewCount+" sits and its max speed is "+maxSpeed+".";
    }
}
```
## Опис класу Car
Клас машини успадовує зміні та методи батьківського класу Transport, завдаяки чому можна лишній раз не писати певні строки коду. У даному класі я перезаписав методи батьківського класу, щоб поміняти всяди слово Transsport на Car, я цее зробив написавши перед функцією @Override. Також із цікавого я додав класу машини зміні на якому паливі машина та колір машини; із методів я додав playMusic.

## Опис класу Airplane
Цей клас ідентичний до класу машин через батька транспорт, від якого наслудіє більшість інформації. До цього класу я додав зміні розмах крил, вмісткість(яку вагу може тримати літак) та максимальна висота.
